# Space Rocket Game

## Project Overview

Space Rocket Game is an interactive game developed using PictoBlox. The player controls a rocket and navigates through obstacles in a space-themed environment. The project demonstrates game development concepts such as sprite movement, collision detection, and user interaction.

## Features

* Rocket movement controls
* Obstacle avoidance gameplay
* Space-themed graphics and background
* Interactive user experience
* Block-based programming using PictoBlox

## Technologies Used

* PictoBlox
* Block Programming
* Scratch-based Game Development

## Project Structure

* `project.sb3` – Main PictoBlox project file
* `screenshots/` – Screenshots of gameplay and code blocks
* `README.md` – Project documentation

## How to Run

1. Download the project file (`project.sb3`).
2. Open the file in PictoBlox.
3. Click the Green Flag to start the game.
4. Use the controls to move the rocket and avoid obstacles.

* Creating interactive applications using PictoBlox
